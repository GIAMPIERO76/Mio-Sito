Attestato Java

Descrizione

Il Linguaggio Java e uno dei Linguaggi di Alto Livello come il C++ usato da tutte le Aziende piccole medie e grandi, per sviluppare tutti i tipi di applicazioni, sia lato Client che lato Server.

Il linguaggio Java come del resto molti altri linguaggi, sviluppano il loro codice in due tipi di modalita':

- sviluppo procedurale

- sviluppo tramite OOP(Programmazione ad Oggetti)

Ambiente di Sviluppo

Per poter sviluppare progetti Java, si ha bisogno di un Software di sviluppo, sul web se ne possono trovare di molti tipi, uno degli ultimi ambienti di sviluppo o IDE piu usati, e Gratuiti e' Eclipse

Sviluppo Procedurale

Per prima cosa creando un progetto Java, importante e la creazione di una Classe, e del Main, tramite il quale Java avvia le sue applicazioni, nel seguente modo:

prima di tutto si mette l'import, che allega al progetto la libreria, che si usera' a seconda del tipo di progetto.

IMPORT JAVA.UTIL.*;

PUBLIC CLASS SCANNER{

PUBLIC STATIC VOID MAIN(STRING ARGS[]){

System.out.println("Hello Java!");

} }

In questo caso avviando l'applicazione, si visualizzera' sul prompt, il messaggio Hello Java!

Le Funzioni

Le Funzioni sono un elemento fondamentale in ogni linguaggio, e anche in Java sono molto usate, ma in cosa consistono le Funzioni,

le funzioni sono come dei contenitori, di istruzioni specifiche, cioe' si danno delle istruzioni a Java in questo modo:

PUBLIC VOID SCRIVI( ){

Istruzioni per la funzione

}

Nelle Funzioni in Java come in altri linguaggi, fondamentali sono le istruzioni decisionali tipo:

IF THEN ELSE

int x = 20;

int y = 18;

if (x > y) {

System.out.println("x is greater than y");

SWITCH CASE BREAK

int day = 4; switch (day) { case 1: System.out.println("Monday"); break; case 2: System.out.println("Tuesday"); break; case 3: System.out.println("Wednesday"); break; case 4: System.out.println("Thursday"); break; case 5: System.out.println("Friday"); break; case 6: System.out.println("Saturday"); break; case 7: System.out.println("Sunday"); break; } // Outputs "Thursday" (day 4)

DO WHILE

int i = 0; do { System.out.println(i); i++; } while (i < 5);

CICLO FOR

for (int i = 0; i <= 10; i = i + 2) { System.out.println(i); }

TRY CATCH //fondamentale per sollevare eccezioni o errori e gestirli a nostro piacere

public class Main { public static void main(String[ ] args) { try { int[] myNumbers = {1, 2, 3}; System.out.println(myNumbers[10]); } catch (Exception e) { System.out.println("Something went wrong."); } } }

ARRAY

String[] cars = {"Volvo", "BMW", "Ford", "Mazda"}; System.out.println(cars[0]); // Outputs Volvo
