Attestato Java

Descrizione

Il Linguaggio Java e uno dei Linguaggi di Alto Livello come il C++ usato da tutte le Aziende piccole medie e grandi, per sviluppare tutti i tipi di applicazioni, sia lato Client che lato Server.

Il linguaggio Java come del resto molti altri linguaggi, sviluppano il loro codice in due tipi di modalita':

- sviluppo procedurale

- sviluppo tramite OOP(Programmazione ad Oggetti)

Ambiente di Sviluppo

Per poter sviluppare progetti Java, si ha bisogno di un Software di sviluppo, sul web se ne possono trovare di molti tipi, uno degli ultimi ambienti di sviluppo o IDE piu usati, e Gratuiti e' Eclipse

Sviluppo Procedurale

Per prima cosa creando un progetto Java, importante e la creazione di una Classe, e del Main, tramite il quale Java avvia le sue applicazioni, nel seguente modo:

prima di tutto si mette l'import, che allega al progetto la libreria, che si usera' a seconda del tipo di progetto.

IMPORT JAVA.UTIL.*;

PUBLIC CLASS SCANNER{

PUBLIC STATIC VOID MAIN(STRING ARGS[]){

System.out.println("Hello Java!");

} }

In questo caso avviando l'applicazione, si visualizzera' sul prompt, il messaggio Hello Java!

Le Funzioni

Le Funzioni sono un elemento fondamentale in ogni linguaggio, e anche in Java sono molto usate, ma in cosa consistono le Funzioni,

le funzioni sono come dei contenitori, di istruzioni specifiche, cioe' si danno delle istruzioni a Java in questo modo:

PUBLIC VOID SCRIVI( ){

Istruzioni per la funzione

}
Nelle Funzioni in Java come in altri linguaggi, fondamentali sono le istruzioni decisionali tipo:

IF THEN ELSE

int x = 20;

int y = 18;

if (x > y) {

System.out.println("x is greater than y");

SWITCH CASE BREAK

int day = 1;

switch (day) {

case 1:

System.out.println("Monday");

break;

} // Outputs "Monday" (day 1)

DO WHILE

int i = 0;

do {

System.out.println(i);

i++;

}

while (i < 5);

CICLO FOR

for (int i = 0; i <= 10; i = i + 2) {

System.out.println(i);

}

Java OOP (Object Operation Program)

La OOP ovvero la programmazione ad oggetti in Java, e' un tipo di programmazione, basato sul concetto di Oggetto.

un Oggetto in Java, viene creato tramite una Classe, che avra' un suo nome(per esemmpio classe persona).

in questo caso nella classe persona, inseriremo, degli attributi(ovvero dei dati per esempio della persona, tipo eta, altezza peso ecc),

e subito dopo i vari metodi, che possono essere tipo (metodo cammina, metodo parla, metodo leggi ecc).

Ma per capire meglio il funzionamento, dela programmazione OOP, cominciamo con degli esempi.

per prima cosa creeremo la classe principale, o classe madre, dove inseriremo tutti i parametri che ci serviranno per operare con gli oggetti

public class persona{

/* ATTRIBUTI */

private string nome;

private string cognome;

private string eta';

/* COSTRUTTORE */

public persona(string nome, string cognome, string eta){

this.nome = nome;

this.cognome = cognome;

this.eta = eta;

/* GETTER SETTER */

public void setNome(String nome) {

this.nome = nome;

}

public String getNome() {

return this.nome;

}

public void setCognome(String cognome) {

this.cognome = cognome;

}

public String getCognome() {

return this.cognome;

}

public void setEta(String eta) {

this.eta = eta;

}

public String getEta() {

return this.eta;

}

}

}
